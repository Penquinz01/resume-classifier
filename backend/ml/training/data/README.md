# Resume Dataset

Place `UpdatedResumeDataSet.csv` in this directory.

## Download

1. Go to [Kaggle — Resume Dataset](https://www.kaggle.com/datasets/gauravduttakiit/resume-dataset)
2. Download `UpdatedResumeDataSet.csv`
3. Place it in this directory (`backend/ml/training/data/`)

## Expected Format

The CSV should have at minimum these columns:

| Column   | Description                    |
|----------|--------------------------------|
| Category | Job role label (e.g., "HR")    |
| Resume   | Raw text content of the resume |
